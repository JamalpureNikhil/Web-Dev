No, a <button> element does not have a border by default in terms of explicit CSS, but it typically appears with a border because browsers apply default styles (user-agent stylesheet) to make buttons visually distinct. 

The exact appearance (including border, padding, background, and font) depends on the browser and operating system. For example, most browsers apply a default border: 2px outset #ccc or similar. 

To remove the default border:

button {
  border: none;
  /* or */
  border: 0;
}

No, a child div does not automatically inherit the parent's height and width in CSS. 

Width: A block-level child div typically takes up the full available width of its parent (due to width: auto), but this is not inheritance—it's default block behavior. 
Height: A child does not inherit parent height.  By default, its height is determined by its own content unless explicitly set.
